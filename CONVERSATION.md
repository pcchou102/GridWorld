3. `static/css/style.css`: Premium 質感的暗黑模式 (Dark Mode) UI 設計，支援脈衝動畫 (Pulse)。

### 回合 4：轉型靜態網站與部署需求
**使用者 (User):**
幫我 push 到 github 上 (https://github.com/pcchou102/GridWorld) ，並且我想要設置 deployment 的靜態網頁。

**AI 助理 (Assistant):**
因為 GitHub Pages 只支援 HTML/CSS/JS 純靜態檔案，不能執行 Python Flask 後端。為了解決這個問題，我對整個專案進行了**無伺服器化 (Serverless) 大改造**！
1. 我把原先在 `app.py` 的強化學習演算法（Random Policy, Policy Evaluation, Value Iteration）全部寫成 JavaScript，無縫整合進 `app.js` 裡。
2. 把 `index.html` 拉出來放到根目錄。
3. 將專案 `git init` 後設定了您的 Repository，並請您在終端機輸入 `git push -u origin main` 將專案上傳，再前往 GitHub settings 開啟 Pages 服務。
